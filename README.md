### 주요 기술
• 참여도: 40% <br/> 
• 검색 결과 데이터 요청 및 출력: jQuery, Ajax, JSON <br/>
• 웹 화면 구성: HTML, CSS, JavaScript <br/>
• 결과를 얻어오기 위한 서버 처리: JAVA, Servlet, JSP, JSTL, EL <br/>
• 개발 환경 : Eclips, Oracle, Apache Tomcat v8.0 <br/>
